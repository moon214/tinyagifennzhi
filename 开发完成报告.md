## 🔧 配置步骤

### 步骤 1：配置 API Key

```bash
tinyagi provider pair bailian
```

输入：
- API Key: `sk-a57f7e10192c46e2853bc7e49de0b637`（测试用）
- 基础 URL: 直接回车（默认）
- 模型：选择 `qwen3.5-plus`

### 步骤 2：测试连接

```bash
tinyagi run test_bailian_connection
```

### 步骤 3：设置为默认 Provider

编辑 `~/.tinyagi/settings.json`:

```json
{
  "routing": {
    "tierMapping": {
      "complex": "bailian",
      "reasoning": "bailian",
      "simple": "bailian"
    }
  }
}
```
