
### ⚠️ 重要说明

**不要使用** `https://coding.dashscope.aliyuncs.com/v1` - 该端点仅支持 Coding Plan 的 Coding Agents

**使用** `https://dashscope.aliyuncs.com/compatible-mode/v1` - OpenAI 兼容模式，支持标准 Chat API

## 🚀 使用命令

### 配置百炼 Provider
```bash
tinyagi provider bailian --model qwen-turbo --api-key sk-xxxxx
```

### 切换模型
```bash
tinyagi model qwen3.5-plus
tinyagi model qwen3-coder-plus
tinyagi model qwen-turbo
```

### 启动 TinyAGI
```bash
tinyagi run
```

## ⚠️ 常见问题

### Q: API Key 无效？
```json
{"error": "Incorrect API key provided"}
```
**解决：** 确保使用标准 API Key（`sk-` 开头），不是 Coding Plan Key（`sk-sp-` 开头）

### Q: 405 Method Not Allowed？
```json
{"error": "Coding Plan is currently only available for Coding Agents"}
```
**解决：** 使用正确的端点 `https://dashscope.aliyuncs.com/compatible-mode/v1`

### Q: 请求超时？
**解决：** 检查网络连接，确保能访问阿里云 API
